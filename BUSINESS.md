# Document Assistant — Бизнес-описание

## Что это

Сервис автоматического анализа страховых запросов. Клиент загружает документ с перечнем требований к страховому покрытию — система возвращает готовый ответ: по каждому пункту указано, есть ли оно в программе страхования, нет или покрывается частично.

---

## Проблема, которую решает

Менеджер страховой компании получает от корпоративного клиента документ на десятки или сотни позиций — список услуг, которые клиент хочет включить в полис. Вручную сопоставить каждый пункт с нормативными программами страхования занимает несколько часов. При этом ошибки неизбежны, а ответ нужен быстро.

---

## Как работает

1. **Клиент присылает файл** в любом формате: Excel-таблица, Word-документ, PDF.
2. **Система читает каждое требование** и сопоставляет его с нормативной базой страховых программ.
3. **Ответ формируется автоматически** в том же формате, что и исходный файл — с тремя добавленными столбцами:
   - **Покрытие по программе** — цитата из нормативного документа
   - **Статус** — Есть / Нет / Частично (с цветовой маркировкой)
   - **Комментарий** — пояснение при частичном покрытии
4. **Итоговое резюме** — выбранная программа, ключевые расхождения, рекомендации.

---

## Ключевые характеристики

| | |
|---|---|
| **Форматы на входе** | .xlsx, .xls, .docx, .doc, .pdf |
| **Формат на выходе** | Тот же, что на входе (оригинал + 3 столбца) |
| **Масштаб** | Документы любого размера — от 10 до 300+ строк |
| **Нормативная база** | Загружается отдельно, легко обновляется |
| **Провайдер ИИ** | Настраивается: Gemini, Anthropic Claude, или локальная Ollama |
| **Развёртывание** | Docker Compose, запускается на одном сервере |

---

## Для кого

- **Страховые компании** — автоматизация ответов на корпоративные тендеры и запросы
- **Страховые брокеры** — быстрое сравнение программ под требования клиента
- **Корпоративные покупатели страховок** — проверка соответствия предложенной программы своему техническому заданию

---

## Что получает бизнес

- Время обработки одного запроса: **от нескольких минут** вместо нескольких часов
- Ответ на **каждое требование** без пропусков и группировок
- Результат **в привычном формате** — не нужно переформатировать вручную
- Нормативная база **обновляется централизованно** — все менеджеры работают с актуальной версией
