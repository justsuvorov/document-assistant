# ВСК ДМС-ассистент — Бизнес-описание

## Что это

Сервис автоматического анализа запросов на ДМС. Корпоративный клиент загружает документ с перечнем требований к страховому покрытию — система возвращает готовый ответ: по каждому пункту указано, есть ли оно в программе страхования, нет или покрывается частично.

---

## Проблема, которую решает

Менеджер страховой компании получает от корпоративного клиента документ на десятки или сотни позиций — список услуг, которые клиент хочет включить в полис ДМС. Вручную сопоставить каждый пункт с нормативными программами занимает несколько часов. При этом ошибки неизбежны, а ответ нужен быстро.

---

## Как работает

1. **Менеджер выбирает нормативную базу** — файл или папка с актуальными программами страхования (`.xlsx`, `.docx`, `.txt`). База читается при каждом запросе, поэтому обновление вступает в силу немедленно.
2. **Менеджер выбирает файл клиента** — Excel-таблица, Word-документ или PDF с требованиями.
3. **Нажимает «Подготовить»** — приложение копирует файлы на сервер и запускает обработку. На экране отображается прогресс-бар с оценкой оставшегося времени.
4. **Система анализирует каждое требование** через LLM, сопоставляя его с нормативной базой. Крупные документы разбиваются на чанки и обрабатываются последовательно.
5. **Ответ формируется в том же формате**, что и исходный файл — с тремя добавленными столбцами:
   - **Покрытие по программе** — цитата из нормативного документа
   - **Статус** — Есть / Нет / Частично (с цветовой маркировкой)
   - **Комментарий** — пояснение при частичном покрытии
6. **Кнопка «Открыть результат»** появляется после завершения — файл открывается сразу.

## Ключевые характеристики

| | |
|---|---|
| **Интерфейс** | Десктопное приложение «ВСК ДМС-ассистент» (Windows) |
| **Форматы на входе** | `.xlsx`, `.xls`, `.docx`, `.doc`, `.pdf` |
| **Формат на выходе** | Тот же, что на входе (оригинал + 3 столбца) |
| **Многолистовые Excel** | Поддерживаются — требования ищутся по всем листам |
| **Масштаб** | Документы любого размера — от 10 до 300+ строк |
| **Нормативная база** | Загружается при каждом запросе — обновление без перезапуска сервера |
| **Провайдер ИИ** | Настраивается: Gemini, Anthropic Claude, или локальная Ollama |
| **Развёртывание** | Docker Compose, запускается на одном сервере |

## Для кого

- **Страховые компании** — автоматизация ответов на корпоративные тендеры и запросы по ДМС
- **Страховые брокеры** — быстрое сравнение программ под требования клиента
- **Корпоративные покупатели страховок** — проверка соответствия предложенной программы своему техническому заданию

---

## Что получает бизнес

- Время обработки одного запроса: **от нескольких минут** вместо нескольких часов
- Ответ на **каждое требование** без пропусков и группировок, с точным сопоставлением по тексту
- Результат **в привычном формате** — не нужно переформатировать вручную
- Нормативная база **обновляется централизованно** — достаточно заменить файл, все следующие запросы используют актуальную версию
